# Agronare Web (React + Vite + Tailwind)

Front-end listo para conectar al backend NestJS del ERP de Agronare.

## Requisitos
- Node.js 20+
- Backend corriendo en http://localhost:3001 (o configura `VITE_API_URL`)

## Comandos
```bash
npm install
npm run dev
# abre http://localhost:5173
```
